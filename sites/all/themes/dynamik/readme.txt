===Dynamik=== 

Theme by Refaktor - http://refaktor.co 
Demo - http://refaktor.co/dynamik_demo

A responsive Drupal 7 theme powered by Twitter Bootstrap. Dynamik features a fully responsive design, easy to use theme options and follows Drupal coding standards. For full theme instructions, please refer to the documentation folder in the root directory of the theme.
