Overview

A responsive Drupal 7 theme powered by Twitter Bootstrap. Dynamik features a fully responsive design, easy to use theme options and follows Drupal coding standards. For theme support that cannot be found in this documentation, please contact me via the ThemeForest author page.

Installation

Option 1:
  • Visit yourdomain.com/admin/appearance/install
  • Upload the dynamik.zip file
  • Enable and set to default
Option 2:
  • Extract the dynamik.zip file and upload via FTP to /sites/all/themes
  • Visit yourdomain.com/admin/appearance/
  • Scroll down to "DISABLED THEMES" and set and enable Dynamik

Menu

To edit your menu visit yourdomain.com/admin/structure/menu (pictured below)

Dynamik uses the Main Menu. Press "List Links" to access the menu overview. From there you can add/remove links, and set the order.

NOTE: If you would like to use Dynamik's built in drop down menu capability, make sure that the parent menu item (the one that indicates the drop down) has the "show as expanded" checkbox enabled.

Theme Settings

Dynamik includes a powerful set of theme settings that allow you to configure many aspects of your site. They are organized in a series of tabbed menu items each with their own set of options. Simply click on the option heading to expand the full set of options. The Dynamik theme options can be accessed by visiting yourdomain.com/admin/appearance/settings/dynamik. All options have default values set, but you should go through each one and adjust them to your liking.

Header

The Dynamik header area allows you to convey a lot of important information to your site's visitors. You can upload your own custom logo, or simply display your site's title and description. You can add a contact phone number or email address, and display any of the following social icons and your social links: Facebook, Twitter, YouTube, Google +, LinkedIn, Vimeo, Pinterest and an RSS feed. To customize your header, visit the theme settings at yourdomain.com/admin/appearance/settings/dynamik and click on the Header tab. From there you can click on any of the options headings and configure each header area to your liking.

Front Page

By default Dynamik uses a custom front page template that features several custom sections including the image slider. All of these sections are fully customizable and can also be disabled. You also have the option to use the standard Drupal homepage by selecting it from the "Select your Front Page template" option.

Image Slider

Dynamik's image slider allows you to upload an image, text caption (can include HTML) and a link for each slide. To enable the slider, visit the Front Page settings tab, click on the Image Slider heading, and check off the "enable image slider" checkbox. Next, chose from either the Elastic Image slider, Nivo slider or Twitter Bootstrap slider. From there, select the number of slides you want to use, and a set of options for each slide will be available. Upload your image, apply your optional caption and/or URL, and save your settings. Your images should be 1170 by 390 for optimal display.

Panels

The panels section is a block region with 4 unique "panels". You can add any type of block you would like to the panels, but if you would like to mimic the default Dynamik panels, you can use the following HTML layout inside your block (be sure to set the block to use the full HTML text format).

<div class="panel_icon">
  <img src="/drupal/sites/all/themes/dynamik_premium/images/colors.png">
</div>
<div class="panel_text">
  <h3 class="panel_title">Theme Options</h3>
    Lorem ipsum dolor sit amet, consectetur adipisicing elit sed do eiusmod tempor
</div>

Highlight

The Highlight section allows you to advertise a specific page or aspect of your site. The settings for the Highlight section are in the Front Page tab of the theme settings. You can enter your own title, text, button text and button URL.

Front Page Region

There is a block region called Front Page that can be used to display any block content you wish. If you want to display your recent blog posts as seen on our demo, you can use the Views module to display your post recent posts. Dynamik uses a custom view template to achieve the view seen on our demo, so you have to call the view Front Page Blog to make sure the template is properly displayed. See the image below for how the view is set up on the demo.

Footer

There are two areas in the Dynamik footer: the main footer (a block region) and a secondary footer which can be used to display copyright information. There are 4 regions inside the main footer, and blocks can be added to them like any other region. You can also choose to not display the footer by disabling it in the Footer theme settings. To configure the secondary footer, also visit the Footer theme settings where you can disable the secondary footer, or enter your own information into the left and right portion of secondary footer (supports HTML).

Popular Tags

Use the Views module to create a block view titled "Popular Tags" that is an HTML list of the taxonomy term "Tags". You can set the number limit to whatever you like, just make sure to name it "Popular Tags" and to make it an HTML list.

Block Regions

Dynamik features 12 block regions where you can add your own custom content. Visit yourdomain.com/admin/structure/block to add your own content to the defined block regions. Press "Add Block" to create a new custom blog, or select an existing block from the Disabled list below the defined block regions.
